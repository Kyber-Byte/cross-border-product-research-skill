# Cross-border Product Research Skill

A free AI skill for turning overseas user comments, reviews, search snippets, and product discussions into a structured cross-border product research report.

中文说明：这是一个免费的跨境选品分析 Skill。你可以把 Reddit、Amazon、TikTok、X、YouTube、论坛、评论区里的用户吐槽和讨论复制进来，它会帮你整理用户痛点、目标人群、商品方向、风险和独立站详情页建议。

## What It Does

- Extracts real user pain points from supplied evidence
- Summarizes target users and usage scenarios
- Reverse-engineers possible product ideas
- Suggests review and TikTok validation angles
- Generates a Shopify product page strategy
- Produces a structured product decision report

## Best For

- Cross-border ecommerce sellers
- Shopify store owners
- Amazon sellers
- TikTok Shop sellers
- Indie makers
- Product researchers

## How To Use

1. Install or copy this skill into your AI coding or agent environment.
2. Give it a product category or scenario.
3. Paste real user evidence, such as comments, reviews, Reddit posts, TikTok notes, or Amazon review snippets.
4. Ask it to generate a product research report.

Example request:

```text
Use this evidence to analyze whether "traveling with dogs" has product opportunities.
Please generate the report in Simplified Chinese.

Evidence:
- Reddit: "I hate leaving my dog in the car during bathroom breaks, but it was unavoidable."
- Amazon review: "The travel bowl leaks in my bag."
- TikTok comment: "My dog gets anxious every time we drive for more than an hour."
```

## Report Sections

The skill outputs these sections:

- `[Sample Judgment]`
- `[Target Segment]`
- `[Top 3 High-Frequency Pain Points]`
- `[Product Reverse Engineering]`
- `[Product Review Suggestion]`
- `[Product Decision]`
- `[Shopify Product Page Strategy]`

## Free Skill vs Hosted Version

This open-source skill is free if you are willing to collect evidence manually.

If you do not want to configure APIs, collect evidence, translate results, or organize evidence by hand, try the hosted version:

http://producttool.nomadwizard.me/

The hosted version can help with automatic evidence collection, translation, candidate evidence browsing, manual confirmation, and report generation.

## Important Notes

- This skill should not invent evidence.
- If fewer than 2 meaningful evidence items are supplied, it should ask for more evidence.
- Final product decisions are directional, not guarantees.
- Always verify compliance, market size, fulfillment, and platform policy before launching a product.
